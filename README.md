# 🛡️ Stellar Shield v3 - Anonymous Feedback Smart Contract

An advanced, decentralized, and completely anonymous feedback system built on top of the **Stellar Network** using **Soroban Smart Contracts**. This repository represents the completion of **Level 3 (Orange Belt)** in the Stellar Monthly developer ecosystem.

## 🚀 Project Overview
Stellar Shield allows users to submit untraceable, secure, and encrypted feedback or ratings within a decentralized ecosystem. This iteration (v3) introduces robust unit testing patterns, optimized state management, and an automated deployment pipeline.

### 🔑 Key Requirements Fulfilled (Level 3)
* **Comprehensive Test Suite:** High-coverage unit tests written in Rust (`cargo test`) verifying state transitions and authorization.
* **CI/CD Integration:** Automated GitHub Actions pipeline for linting, formatting, and building the Soroban WASM target.
* **Workspace Architecture:** Organized multi-contract structure leveraging Cargo workspaces for modularity.
